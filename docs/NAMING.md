# Project Naming

**Selected working name:** **LexPad**

| Candidate | Pros | Cons | Verdict |
|-----------|------|------|---------|
| **LexPad** | Ties to Lexilla; short; no trademark conflict | Generic | ✅ **Selected** |
| Margin | Editor metaphor | Obscure | Backup |
| Scratchpad | Descriptive | Common, hard to SEO | Backup |
| Scribe | Clean | Many existing products | Rejected |
| PadMac | Clear platform | Awkward | Rejected |

## Constraints

- **Do not use:** Notepad++, N++, frog/chameleon imagery, or Don Ho biography
- **License:** GPL-3.0 recommended if linking Scintilla statically; confirm before release
- **Domain:** Check `lexpad.app`, `lexpad.dev`, GitHub org availability before public launch
