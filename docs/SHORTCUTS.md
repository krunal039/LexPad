# LexPad Keyboard Shortcuts

Default macOS-style shortcuts. Switch to **Notepad++ Compatible** in **Settings → Shortcuts** to map Ctrl-based bindings (Ctrl+F, Ctrl+H, etc.).

## File

| Shortcut | Action |
|----------|--------|
| `Cmd+T` | New tab |
| `Cmd+Shift+N` | New window |
| `Cmd+O` | Open… |
| `Cmd+P` | Quick Open |
| `Cmd+S` | Save |
| `Cmd+Shift+S` | Save As… |

## Edit / Search

| Shortcut | Action |
|----------|--------|
| `Cmd+Z` / `Cmd+Shift+Z` | Undo / Redo |
| `Cmd+F` | Find |
| `Cmd+Option+F` | Replace |
| `Cmd+Shift+F` | Find in Files |
| `Cmd+Shift+R` | Replace in Files |
| `Cmd+L` | Go to Line |
| `Cmd+E` | Incremental search |
| `Cmd+Shift+P` | Command palette |
| `Cmd+/` | Toggle line comment |
| `Cmd+Ctrl+D` | Add next occurrence |
| `Ctrl+Space` | Show completions |

## View

| Shortcut | Action |
|----------|--------|
| `Cmd+,` | Settings |

## Navigation

| Shortcut | Action |
|----------|--------|
| `F2` | Toggle bookmark |
| Next / Previous Bookmark | Edit menu (assign in Shortcuts prefs) |

## Multi-cursor and column mode

| Action | Shortcut / gesture |
|--------|-------------------|
| Add cursor | `Cmd+Click` |
| Column selection | `Option+drag` |
| Column editor | Tools → Column Editor… |

## Tab bar (mouse)

| Action | Gesture |
|--------|---------|
| New tab | Double-click empty tab bar area |
| Rename tab | Double-click tab title |
| Open tabs menu | Click **▼** on tab bar |
| Split by drag | Drag tab to editor edge |

## Notepad++ preset (Settings → Shortcuts)

When **Notepad++ Compatible** is selected, common mappings include:

| N++ style | LexPad command |
|-----------|----------------|
| `Ctrl+F` | Find |
| `Ctrl+H` | Replace |
| `Ctrl+G` | Go to Line |
| `Ctrl+S` | Save |
| `Ctrl+Shift+S` | Save As |
| `Ctrl+N` | New tab |
| `Ctrl+W` | Close tab (if mapped) |

Customize any command in **Settings → Shortcuts**. Conflicts are highlighted; reset restores defaults.

## Resizable panels

Drag the divider between:

- Editor split panes (horizontal / vertical)
- Left dock panels (workspace, vertical tab bar, etc.)
- Right dock panels (document map, function list, Git, etc.)

No keyboard shortcut — use the mouse on panel edges.
